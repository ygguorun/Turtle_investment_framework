# 写作风格要求（所有 Agent 共享）

1. **结论先行**：每个维度开头用 **一句话加粗结论** 引导读者，分析论证放在后面
2. **重点突出**：关键数据和评级用 **加粗** 标识，不要让关键信息淹没在大段文字中
3. **格式选择**：结构化对比数据用表格，分析论述用段落，不要把所有内容都堆进表格
4. **逻辑过渡**：段落之间有清晰的因果/递进关系，避免信息跳跃
5. **去重**：维度之间不重复引用同一数据或结论，交叉引用时简短提及即可
6. **通俗表达**：用通俗语言解释行业术语和专业判断，让非专业投资者也能理解核心逻辑
7. **金额单位**：百万元，千位逗号分隔
8. **利润口径**：使用 GAAP 归母净利润（除非 preflight 指定其他口径）
