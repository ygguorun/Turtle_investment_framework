# 框架适用范围与局限性声明

> 本文件记录定性分析框架的适用范围、已知局限性，以及需要人工覆核的场景。
> 分析 Agent 在遇到这些场景时应主动标注，而非尝试自动修正。

---

## 一、框架最适用的公司类型

- 单一主业或主业占比 > 70% 的公司
- 商业模式清晰、可用一句话描述的公司
- 有至少 5 年连续可比财务数据的公司
- 行业竞争格局相对稳定、可对标的公司

## 二、框架效力降级的场景

| 场景 | 降级原因 | 应对策略 |
|------|---------|---------|
| **大型多元化集团**（收入来源 > 3 个不相关行业） | 异质性高，AI 难以同时深入分析多个行业的竞争格局 | 标注"⚠️ 高异质性公司"，对每个业务板块单独给出置信度，整体结论标注为参考性 |
| **投资收益主导型**（invest_income / 利润总额 > 30%） | PE 被非经营性收入压低，PB 可能被公允价值计量扭曲 | 标注 PE/PB 失真风险，建议扣除投资收益后重算核心 PE，或使用 SOTP |
| **控股平台型公司**（如多个上市子公司） | 合并报表掩盖子公司价值分布 | 触发 D6 SOTP 分析，但标注 SOTP 精度受限于子公司信息披露 |
| **转型期公司**（商业模式正在重大变化） | 历史数据外推性差 | 标注"⚠️ 转型期，历史数据参考价值有限"，降低定量分析权重 |
| **财务数据不足 3 年**（新上市/借壳） | 趋势分析不可靠 | 降级标注，缩短对标周期 |
| **强周期顶部/底部** | 当前盈利不代表正常化水平 | 需配合 D3 周期判断 + 正常化盈利估算 |

## 三、需要人工覆核的判断

以下判断 AI 可以给出初步评估，但建议投资者自行验证：

1. **护城河可持续性**：AI 基于历史数据判断，但技术颠覆、政策变化等非线性事件需人工判断
2. **管理层诚信度**：AI 只能分析公开信息中的信号，内部治理问题往往在暴露前无法从报表中发现
3. **SOTP 估值精度**：非上市子公司的估值高度依赖假设
4. **投资收益的可持续性**：联营公司分红能力取决于其自身经营状况，需逐一分析
5. **会计政策选择的影响**：如权益法 vs 公允价值计量、资本化 vs 费用化，这些选择对利润和净资产的影响需要专业判断

## 四、何时不应单独依赖本框架

- 利润中超过 50% 来自投资收益或公允价值变动
- 公司涉及 4 个以上不相关行业
- 公司正在经历重大并购/分拆/资产重组
- 最近 2 年管理层全面更换

这些情况下，本框架的输出应仅作为人工深度分析的起点，而非结论。
