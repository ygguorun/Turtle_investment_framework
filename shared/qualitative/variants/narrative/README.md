# 叙事增强版 Prompt（Narrative Variant）

本目录保存了定性分析模块的**叙事增强版**prompt，与默认版（`shared/qualitative/`）的区别：

| 改动点 | 默认版 | 叙事增强版 |
|--------|--------|-----------|
| D2.3 护城河来源 | Bullet 列表 | "起点→深化→结果"因果链段落 |
| D2.5 竞争对手 | 一段话概括可持续性 | 逐对手叙事+追赶难度表 |
| 总结"商业模式本质" | "用通俗语言概括（1段）" | 要求大白话一句话+因果叙事展开 |
| 写作风格 | 6条规则 | 7条（新增"叙事连贯性"） |
| judgment_examples | 5节 | 8节（新增逻辑链示例、对手可持续性示例、商业本质示例） |

## 背景

通过对太阳纸业（002078.SZ）的定性报告与一份外部高质量研究报告进行对比分析，发现默认版在以下方面存在改进空间：

1. **叙事连贯性**：默认版报告读起来像结构化检查清单，信息密度高但缺乏因果推演的"故事感"；外部报告则像一篇连贯的投资研究论文
2. **护城河论证说服力**：默认版用 bullet 列举护城河要素（资源独占、垂直整合、自给率…），信息完整但读者无法感受优势如何层层累积
3. **竞争对手分析深度**：默认版的"优势差距可持续性"仅一句话概括，未区分不同对手的追赶难度和时间窗口
4. **商业本质表达**：默认版总结部分用术语化语言（"林浆纸一体化商业模式"），缺乏让非专业读者脑中形成画面的大白话表达

叙事增强版在**保留默认版全部结构化优势**（数据锚定、量化验证、虚假优势辨析、监控锚点、结构化参数输出）的前提下，针对以上 4 点进行了增量式优化。

## 具体改动详情

### P1: D2.3 护城河来源 — 逻辑链叙事

在 `qualitative_assessment.md` 步骤3 中新增"护城河来源叙事（Logic Chain）"指令块，要求对评级"强"或"较强"的 Greenwald 维度用三步因果链叙事：

- **起点**（锁定）：企业锁定了什么稀缺资源或关键位置？
- **深化**（构建）：如何发展为不可复制的结构性优势？
- **结果**（成果）：形成了什么可量化的经济成果？

评级为"中等"或"弱"的维度保留评级+一句话说明，无需展开。

`judgment_examples.md` 新增 §2b 对比示例（好的逻辑链写法 vs 差的纯列表写法）。

### P2: D2.5 竞争对手 — 逐对手可持续性分析

将原来一行的"(4) 优势差距可持续性"扩展为：
- 逐对手追赶难度表（对手 × 领先领域 × 追赶难度 × 时间窗口）
- 每个对手 1 段叙事，解释差距为何可持续、核心约束是什么
- 维度对比表建议增加 1-2 个行业特定维度，"核心差异原因"栏要求 ≥2 句话

`judgment_examples.md` 新增 §5b 逐对手叙事示例。

### P3: 总结"商业模式本质" — 大白话叙事

将 `qualitative_assessment.md` 和 `agent_summary.md` 中的"用通俗语言概括（1段）"替换为更明确的指令：
- 先写一句不含行业术语的**大白话商业本质**（加粗），用动词和具体事物让读者脑中形成画面
- 然后 1-2 段因果叙事展开：护城河本质 → 竞争对手为什么做不到 → 代价是什么

`judgment_examples.md` 新增 §6 "商业本质一句话"示例（好的画面感写法 vs 差的标签化写法，含太阳纸业、长江电力、贵州茅台三个案例）。

### P4: 全局写作风格 — 叙事连贯性

在 `qualitative_assessment.md` 和 `writing_style.md` 中：
- 增强规则 4（逻辑过渡）：要求使用明确的因果/递进/转折过渡句，读者应感受到"现象→原因→结论→启示"的推进方向
- 新增规则 7/9（叙事连贯性）：分析论述用散文段落而非 bullet 列表，数据服务于论点（先判断后论证）

### P5: Summary Agent 竞争定位

`agent_summary.md` 竞争定位段落从"在行业中的位置（1-2段）"扩展为结构化 2 段要求：
- 第 1 段：逐对手核心差异 + 差距方向（扩大/缩小）
- 第 2 段：产业链地位变迁 + 具体里程碑佐证

## 使用方式

如需使用叙事增强版，将本目录下的文件替换 `shared/qualitative/` 中对应文件即可。

## 效果对比

以太阳纸业（002078.SZ）为测试案例，对比报告见 `output/002078_太阳纸业/version_comparison.md`：
- v1（默认版）：`output/002078_太阳纸业/qualitative_report_v1.md`
- v2（叙事增强版）：`output/002078_太阳纸业/qualitative_report_v2.md`

核心结论（护城河评级"较强"）一致，但叙事增强版在以下方面有明显改进：
- 护城河论证从 bullet 列表升级为因果链段落
- 竞争对手分析从 5 维扩展至 7 维，新增逐对手可持续性叙事
- 商业本质表达更具画面感和记忆点
- 部分评级因论证更充分而上调（可持续性：中等→高，证据强度：中等→强）

**注意**：叙事增强版生成的报告篇幅更长、阅读时间更长。偏好简洁检索式阅读的用户建议使用默认版。
